HOW TO MAKE NEW BAND?<br>
MAKE AN ITEM, AND SET ITEM.BAND TO PROPER STRING<br>
like: ITEM.band = "blue"<br>
Now add key to the table inside cl_hooks.lua<br>
like:<br>
[string identificator={Color Color,string description}]<br>
